# money tracker

This is a money tracker. 
It counts money from phone messages.

### What it does
It reads a file with messages. 
It looks for numbers and the word RWF.
It finds how much money was received. 
It finds how much money was sent. 
Then it shows the total amounts.

### How to use it
1. Put messages in the file called sms_data.xml.
2. Run the code using Python.
3. The totals will show up on the screen.

### Why it is helpful
It makes counting money fast. 
It helps people see where money goes.
It is a simple tool for everyone.
